# Application

## Basic usage

Illustration application. It suggests tweets to a user.
To run it, just launch the server.py script.

## Dependencies

* Python 2
* Flask >= 0.10
* NLTK >= 3.2
* NumPy >= 1.6.1
* Preprocessor >= 1.1.3
* Pylab >= 0.1.3
* SciPy >= 0.9
* Scikit-Learn (sklearn) >= 0.17.1
* tweet-preprocessor >= 1.1
* Tweepy >= 3.5.0
